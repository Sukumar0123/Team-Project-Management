# TeamFlow Board - GitHub Style Team Task Tracker

A premium frontend-only project management app built with **HTML, CSS, and JavaScript**.

## Features

- GitHub-style project board
- Todo, In Progress, Review, Done columns
- Add, edit, delete tasks
- Assign tasks to team members
- Add/edit/delete team members
- Automatically unassign tasks when a member is deleted
- Unassigned tasks filter
- Member workload dashboard
- Project progress dashboard
- Project archive option
- Labels and priorities
- Task comments / work notes timeline
- Activity history
- Drag and drop tasks
- Search and filters
- Dark/light theme
- LocalStorage persistence
- Responsive SaaS-style glassmorphism UI

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Open Vercel.
3. Import the GitHub repository.
4. Framework preset: **Other**.
5. Build command: leave empty.
6. Output directory: leave empty or use `.`.
7. Deploy.

## Local Run

Open `index.html` directly in the browser.
